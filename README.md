# Analisador de Texto em C#

Um analisador de texto simples para contar palavras, frequência e frases mais usadas, desenvolvido em C# com WPF.

## Funcionalidades

- Contagem de palavras
- Cálculo de frequência de palavras

## Como Executar

1. Clone o repositório: `git clone https://github.com/[SEU_USUARIO]/AnalisadorDeTextoCSharp.git`
2. Entre na pasta do projeto: `cd AnalisadorDeTextoCSharp`
3. Execute o projeto WPF: `dotnet run --project AnalisadorDeTexto.WPF/AnalisadorDeTexto.WPF.csproj`

## Estrutura do Projeto

- **AnalisadorDeTexto.Core:** Biblioteca de classes com a lógica principal da análise de texto.
- **AnalisadorDeTexto.WPF:** Aplicação de desktop com a interface gráfica.
- **AnalisadorDeTexto.Tests:** Projeto de testes unitários.
